# Statistical Distribution Analysis Project

## Project Overview
This project analyzes transaction data using statistical distributions and visualization techniques.
The goal is to understand data behavior, distribution shape, and normality.

## Dataset
The dataset contains:
- transaction_amount
- transaction_count
- other financial related variables

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Analysis Performed
- Data Loading
- Data Cleaning
- Statistical Distribution Analysis
- Binomial Distribution
- Poisson Distribution
- Log Normal Distribution
- Power Law Distribution
- Q-Q Plot
- Box-Cox Transformation
- Z-Score
- PDF Plot

## Output
- Distribution charts
- Q-Q Plot
- Statistical interpretation
- Conclusion
