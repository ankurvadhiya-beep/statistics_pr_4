# Project Summary

This project performs statistical distribution analysis on transaction dataset.

Steps:
1. Load dataset using pandas
2. Check basic dataset information
3. Analyze statistical distributions
4. Apply Binomial distribution
5. Apply Poisson distribution
6. Apply Lognormal and Power law
7. Generate Q-Q plot
8. Apply Box-Cox transformation
9. Calculate Z-score
10. Plot Probability Density Function

Result:
- Data is right skewed
- Data not normally distributed
- Transformation improves normality
- Outliers detected using Z-score
